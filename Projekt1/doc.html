Dokumentacja - projekt 1

Michał Kuś

1. Wymagania

  Wybrane zagadnienie z matematyki, fizyki czy informatyki zaprezentowane z wykorzystaniem graficznych animacji (symulacji) zrealizowanych z wykorzystaniem języka JavaScript i CSS 3.0.

2. Temat

  Animacje przedstawiające trójkąt Sierpińskiego, zbiór Julii oraz zbiór Mandelbrota

3. Lokalizacja projektu

  Projekt został wrzucony na serwer Pascal oraz na IBM Cloud

4. Kod

  Projekt został wykonany w technologiach HTML5, CSS 3.0 z wykorzystaniem Sass, oraz JavaScript. Kod źródłowy poszczególnych animacji znajduje się w plikach .js z odpowiednimi nazwami. Projekt przetestowano w przeglądarkach Opera GX, Google Chrome i Microsoft Edge

5. Źródła

Zbiór Mandelbrota
Zbiór Julii
Trójkąt Sierpińskiego
